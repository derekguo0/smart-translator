# 🧪 翻译功能测试说明

## 📋 测试步骤

### 1. 启动应用
访问：http://localhost:8000

### 2. 测试文本翻译

#### 简单测试：
- **输入文本**：`Hello, how are you today?`
- **选择语言**：英文 → 中文
- **预期结果**：`你好，你今天好吗？`

#### 长文本测试：
```
The quick brown fox jumps over the lazy dog. This is a common English sentence that contains every letter of the alphabet. It's often used to test keyboards, fonts, and other text-related systems. The sentence is also known as a pangram, which is a sentence that uses every letter of the alphabet at least once.
```
- **选择语言**：英文 → 中文
- **预期结果**：应该返回完整的中文翻译，不会被截断

#### 中文测试：
```
今天天气很好，我们去公园散步吧。公园里有很多花草树木，空气清新，非常适合散步和放松心情。
```
- **选择语言**：中文 → 英文
- **预期结果**：完整的英文翻译

### 3. 测试OCR图片翻译

#### 准备测试图片：
1. 创建一个包含文字的图片（截图、拍照等）
2. 确保图片中的文字清晰可见
3. 推荐使用包含中英文的图片

#### 测试步骤：
1. 点击上传区域或直接拖拽图片
2. 等待OCR识别（会显示进度）
3. 查看识别出的文字是否正确
4. 选择目标语言
5. 点击翻译按钮
6. 检查翻译结果是否完整

### 4. 测试剪贴板功能

#### 文本粘贴：
1. 复制一段文字到剪贴板
2. 点击应用中的"粘贴"按钮
3. 确认文字被正确粘贴

#### 图片粘贴：
1. 截取一张包含文字的屏幕截图
2. 在翻译应用中按 `Ctrl+V` (Windows) 或 `Cmd+V` (Mac)
3. 确认图片被正确粘贴并显示预览

## 🚨 常见问题解决

### 如果翻译失败：
1. 检查网络连接
2. 尝试更换翻译语言对
3. 减少文本长度
4. 刷新页面重试

### 如果OCR识别不准确：
1. 确保图片清晰
2. 选择正确的源语言
3. 使用对比度高的图片
4. 避免手写文字

### 如果功能无响应：
1. 打开浏览器开发者工具查看错误
2. 检查是否有网络请求被阻止
3. 尝试使用不同的浏览器
4. 清除浏览器缓存
