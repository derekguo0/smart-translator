# 🌐 智能翻译应用

一个功能强大的网页翻译工具，支持文本翻译和图像OCR翻译，提供Google翻译和ChatGPT AI翻译两种模式。

## ✨ 主要特性

### 🎯 双模式翻译
- **🌐 Google翻译**：免费、快速、稳定，支持100+种语言
- **🤖 ChatGPT翻译**：AI智能翻译，支持上下文理解，质量极高

### 📱 多种输入方式
- **文本输入**：直接输入或粘贴文本进行翻译
- **图像OCR**：上传图片自动识别文字并翻译
- **拖拽上传**：支持拖拽图片到翻译区域
- **剪贴板粘贴**：Ctrl+V 直接粘贴图片

### 🎨 优秀的用户体验
- **响应式设计**：完美适配桌面和移动设备
- **首屏可用**：翻译功能在首屏即可操作
- **实时反馈**：详细的进度提示和状态显示
- **历史记录**：自动保存翻译历史，支持快速查看

## 🚀 快速开始

### 启动应用

```bash
# 克隆项目
git clone [项目地址]
cd 翻译应用

# 启动本地服务器
python3 -m http.server 8000

# 访问应用
在浏览器中打开: http://localhost:8000
```

### 基础使用

1. **Google翻译模式**（默认）
   - 选择"Google翻译"
   - 输入文本或上传图片
   - 点击"开始翻译"
   - 完全免费，无需配置

2. **ChatGPT翻译模式**（高级）
   - 点击"配置ChatGPT"按钮
   - 输入OpenAI API密钥
   - 选择"ChatGPT翻译"
   - 享受AI智能翻译

## 🔧 配置ChatGPT翻译

### 获取API密钥

1. 访问 [OpenAI API Keys](https://platform.openai.com/api-keys)
2. 注册账户并验证邮箱
3. 创建新的API密钥
4. 复制密钥（格式：sk-...）

### 在应用中配置

1. 点击"配置ChatGPT"按钮
2. 输入API密钥
3. 保存配置
4. 选择"ChatGPT翻译"开始使用

详细配置说明请查看：[ChatGPT翻译配置说明.md](ChatGPT翻译配置说明.md)

## 🎨 界面设计

### 简洁布局
- **左侧**：翻译控制（质量选择、输入区域、图片上传）
- **右侧**：翻译结果（结果显示、OCR文本、操作按钮）
- **底部**：翻译历史（网格视图、搜索管理）

### 视觉特色
- **现代化UI**：使用Tailwind CSS + FontAwesome图标
- **色彩主题**：绿色代表Google，紫色代表ChatGPT
- **交互反馈**：悬停效果、加载动画、状态提示

## 🔍 技术特性

### OCR识别
- **多语言支持**：基于Tesseract.js引擎
- **智能语言检测**：自动选择合适的OCR语言
- **格式保持**：保留原文的换行和格式

### 数据安全
- **本地存储**：翻译历史和配置仅保存在浏览器
- **隐私保护**：API密钥不会上传到服务器
- **可控删除**：支持清除历史和配置

### 错误处理
- **网络重试**：自动重试失败的请求
- **降级机制**：ChatGPT失败时自动切换到Google
- **友好提示**：详细的错误信息和解决建议

## 🎯 使用技巧
