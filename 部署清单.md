# ✅ 部署清单

## 🎯 部署前准备

### 基础环境
- [ ] 安装Git
- [ ] 安装Node.js（版本14+）
- [ ] 选择部署平台

### 账号注册
- [ ] GitHub账号（必须）
- [ ] Vercel账号（推荐）
- [ ] Netlify账号（可选）
- [ ] Firebase账号（可选）

## 📋 部署步骤

### 选择部署方式

#### 方式1：自动化脚本部署
- [ ] 下载项目到本地
- [ ] 打开终端/命令提示符
- [ ] 执行部署脚本
  - macOS/Linux: `./deploy.sh vercel`
  - Windows: `deploy.bat vercel`
- [ ] 按照提示完成部署

#### 方式2：手动部署
- [ ] 上传代码到GitHub
- [ ] 在部署平台连接GitHub仓库
- [ ] 点击部署按钮
- [ ] 等待部署完成

## 📞 遇到问题时

### 常见问题排查
1. **部署失败**
   - 检查网络连接
   - 确认账号权限
   - 重试部署命令

2. **网站无法访问**
   - 检查DNS设置
   - 确认部署状态
   - 等待几分钟生效

3. **翻译功能不工作**
   - 检查API密钥
   - 确认API配额
   - 查看浏览器控制台
